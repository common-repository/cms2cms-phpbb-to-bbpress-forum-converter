Plugin for automated data migration from phpBB to bbPress. Convert phpBB to bbPress easily and fast.


== Description ==

With the help of CMS2CMS, you will be able to move your phpBB forum data to bbPress. Save your time with automated forum content migration.

= HOW THIS PLUGIN HELPS: =

Free Demo Version is available to see how the plugin works (transfers limited amount of data from one forum platform to another);
The price depends on the total amount of the entities you want to migrate;
15 min for an average migration;
Availability of live help. If you have any migration related questions, our tech team will try to answer them;
If you want to perform a customized migration, make a request for a CMS2CMS data migration service package.

= WHAT WILL BE MOVED FROM PHPBB TO bbPress? =

* Forums/Categories;
* Forum Posts;
* Forum Topics;
* Forum Users;
* Forum Images;
* Forum Attachments;

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the  “CMS2CMS:  Automated phpBB To bbPress Migration Pluginn”.
5. Register an account or sign in to the existing one.
6. Provide your Existing forum URL address.
7. Download the Connection Bridge and place it into the Existing forum root folder. You can find a detailed instruction here.
8. Go back to the plugin and click “Verify Connection”. You’ll be redirected to the CMS2CMS website to proceed with your migration.
